# COVID-19 Data Analysis

## Internship
Tech-Driven Internship Program in Data Science – Kodbud

## Objective
Analyze COVID-19 data using Python and Data Science techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- Country/Region
- Confirmed Cases
- Deaths
- Recovered Cases
- Active Cases
- WHO Region

## Analysis Performed
1. Top Countries by Confirmed Cases
2. Death Analysis
3. Active Cases Analysis
4. Recovery Rate Analysis
5. WHO Region Analysis
6. Correlation Heatmap

## Key Findings
- Countries with more confirmed cases generally had more deaths.
- Recovery rates varied across countries.
- Some WHO regions were affected more than others.
