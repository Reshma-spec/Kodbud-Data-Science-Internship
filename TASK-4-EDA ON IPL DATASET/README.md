# IPL Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on the Indian Premier League (IPL) dataset using Python.

## Objectives
- Analyze the most successful IPL teams
- Explore stadium hosting trends
- Study toss decision patterns
- Identify top Player of the Match award winners
- Analyze season-wise match statistics
- Visualize insights using charts and graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Most winning teams in IPL history
- Popular venues hosting IPL matches
- Toss decision preferences
- Top Player of the Match performers
- Match distribution across seasons

## Project Structure
- Data Cleaning
- Data Exploration
- Statistical Analysis
- Data Visualization
- Insights and Conclusion

## Outcome
This project demonstrates practical data analysis skills and visualization techniques for extracting meaningful insights from sports datasets.
