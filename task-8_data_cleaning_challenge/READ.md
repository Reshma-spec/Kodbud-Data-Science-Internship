# Customer Behavior Analysis

## Internship

Tech-Driven Internship Program in Data Science – Kodbud

## Objective

Analyze customer behavior data to identify purchasing patterns, customer preferences, and key business insights.

## Dataset Description

The dataset contains customer information such as:

* Customer ID
* Age
* Gender
* Purchase Amount
* Product Category
* Frequency of Purchases
* Customer Ratings

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

### 1. Data Cleaning

* Checked missing values
* Removed duplicates
* Verified data types

### 2. Exploratory Data Analysis (EDA)

* Customer age distribution
* Gender distribution
* Purchase amount analysis
* Product category analysis

### 3. Customer Insights

* Most popular product categories
* High-spending customer groups
* Purchase behavior trends
* Customer segmentation observations

### 4. Visualizations

* Bar charts
* Histograms
* Pie charts
* Correlation heatmaps

## Key Findings

* Identified customer groups with higher spending behavior.
* Determined the most frequently purchased product categories.
* Observed purchasing trends across different demographics.
* Generated actionable business insights from customer data.

## Conclusion

Customer behavior analysis helps businesses understand customer preferences and improve marketing and sales strategies.
