# Salary Prediction Model using Linear Regression

## Overview

This project predicts employee salaries using a Linear Regression model. The dataset contains employee details such as designation, age, experience, ratings, leaves, and other attributes.

## Objectives

* Build a salary prediction model.
* Apply Linear Regression.
* Evaluate model performance using Mean Squared Error (MSE) and R² Score.
* Visualize prediction results.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Workflow

1. Data Loading
2. Data Preprocessing
3. Feature Encoding
4. Train-Test Split
5. Linear Regression Model Training
6. Salary Prediction
7. Model Evaluation
8. Data Visualization

## Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score

## Visualizations

* Actual vs Predicted Salary Plot
* Residual Plot
* Correlation Heatmap

## Results

The Linear Regression model successfully predicts employee salaries based on available employee information and demonstrates the use of regression techniques in machine learning.

## Conclusion

This project provides a practical introduction to salary prediction using supervised machine learning and highlights the importance of data preprocessing, feature engineering, and model evaluation.
